# Google Sheets API

> The Google Sheets API (v4) lets you read, write, and format data in Google
> Sheets programmatically over REST/JSON. It supports creating spreadsheets,
> reading and writing cell values, batch value operations, structural and
> formatting updates via batchUpdate, sheet management, data filters, and
> developer metadata. Auth is OAuth 2.0 (Google Identity) with granular
> Sheets/Drive scopes; public read of published sheets can use an API key.

When installing Google client libraries, always check the registry for the
latest version rather than relying on memorized version numbers. For Python use
`google-api-python-client`, for Node.js `googleapis`, for Go
`google.golang.org/api/sheets/v4`. Never hardcode an old version from training
data — verify the current version first.

## APIs
- [Google Sheets API v4](https://developers.google.com/workspace/sheets/api): REST API to read, write, and format Google Sheets data. Base URL https://sheets.googleapis.com/v4.
- [Google Apps Script Spreadsheet Service](https://developers.google.com/apps-script/reference/spreadsheet): Built-in Apps Script service to create, access, and modify Sheets files server-side.
